# 实验 6：四个子网划分与路由配置操作指南（八台 PC 版）

## 二、实验软件和设备

实验中需要使用的设备修改为：

| 设备类型 | 数量 | 说明 |
|---|---:|---|
| 路由器 Router | 3 台 | 推荐使用 `2911`，也可以使用 `1941` |
| 交换机 Switch | 4 台 | 推荐使用 `2960` |
| PC 电脑 | 8 台 | 每个部门放 2 台测试主机 |
| 直通线 Copper Straight-Through | 若干 | 用于连接 PC、交换机和路由器 |

设备命名建议修改为：

| 设备 | 名称 |
|---|---|
| 路由器 | R1、R2、R3 |
| 交换机 | S1、S2、S3、S4 |
| 电脑 | PC1、PC2、PC3、PC4、PC5、PC6、PC7、PC8 |

## 五、实验拓扑设计

本实验仍然使用 3 台路由器连接 4 个局域网，但每个局域网中放置 2 台 PC。

逻辑结构如下：

```text
部门 1：PC1、PC2 -- S1 -- R1
部门 2：PC3、PC4 -- S2 -- R1、R2
部门 3：PC5、PC6 -- S3 -- R2、R3
部门 4：PC7、PC8 -- S4 -- R3
```

更具体地说：

```text
部门 1：PC1、PC2 接入 S1，S1 接入 R1
部门 2：PC3、PC4 接入 S2，S2 同时连接 R1 和 R2
部门 3：PC5、PC6 接入 S3，S3 同时连接 R2 和 R3
部门 4：PC7、PC8 接入 S4，S4 接入 R3
```

## 九、配置路由器

路由器配置不需要改变。

R1、R2、R3 的接口 IP 地址和静态路由仍然使用原来的配置。

---

## R1 完整配置

```text
enable
configure terminal
hostname R1

interface gigabitEthernet 0/0
ip address 218.197.17.1 255.255.255.192
no shutdown
exit

interface gigabitEthernet 0/1
ip address 218.197.17.65 255.255.255.192
no shutdown
exit

ip route 218.197.17.128 255.255.255.192 218.197.17.66
ip route 218.197.17.192 255.255.255.192 218.197.17.66

end
write
```

## 十五、测试跨局域网通信

可以在 PC1 上测试访问其他部门的 PC。

在 PC1 的 Command Prompt 中输入：

```text
ping 218.197.17.70
ping 218.197.17.80
ping 218.197.17.140
ping 218.197.17.150
ping 218.197.17.200
ping 218.197.17.210
```

也可以在 PC8 上测试访问其他部门的 PC：

```text
ping 218.197.17.10
ping 218.197.17.20
ping 218.197.17.70
ping 218.197.17.80
ping 218.197.17.140
ping 218.197.17.150
```

如果都能收到类似下面的结果：

```text
Reply from 218.197.17.x
```

说明 4 个局域网之间已经实现互相通信。

第一次 `ping` 可能会出现一次超时，这是因为设备需要学习 ARP 地址，属于正常现象。可以再 `ping` 一次。

完成配置后，应达到以下效果：

1. PC1 可以 ping 通 PC2。

3. PC5 可以 ping 通 PC6。

6. PC3、PC4 可以 ping 通自己的网关 `218.197.17.65`。
7. PC5、PC6 可以 ping 通自己的网关 `218.197.17.129`。
8. PC7、PC8 可以 ping 通自己的网关 `218.197.17.193`。
9. PC1 可以 ping 通 PC3、PC4、PC5、PC6、PC7、PC8。
10. PC8 可以 ping 通 PC1、PC2、PC3、PC4、PC5、PC6。
11. 4 个部门所在的 4 个局域网之间可以互相通信。

## 十九、实验结论

本实验将 `218.197.17.0/24` 平均划分成 4 个 `/26` 子网。

每个子网地址数量为 64 个，其中可用主机地址为 62 个。

本实验将原来的 4 台 PC 改为 8 台 PC，每个部门放置 2 台 PC：

| 部门 | PC | IP 地址 |
|---|---|---|
| 部门 1 | PC1、PC2 | 218.197.17.10、218.197.17.20 |
| 部门 2 | PC3、PC4 | 218.197.17.70、218.197.17.80 |
| 部门 3 | PC5、PC6 | 218.197.17.140、218.197.17.150 |
| 部门 4 | PC7、PC8 | 218.197.17.200、218.197.17.210 |

通过 3 台路由器连接 4 个局域网，并配置路由器接口 IP 地址和静态路由后，8 台 PC 可以实现同部门通信和跨部门通信。
